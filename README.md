# Getting Started

## Install

```bash
npm install
```

Create a folder called `project` public/images/


## Required packages
```bash
# mac
brew install gs graphicsmagick

# linux
sudo apt-get install graphicsmagick
sudo apt-get install imagemagick
```

## Run dev server; 

```bash
npm run dev
# or
yarn dev
```
